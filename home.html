Sliding Puzzle

Instructions

This game is about one of the ancient advanced civilised city's exploration "DWARAKA"
Here these are the game rules:
1. Each image represents a level respectively such as image1 -> level1, image2 -> level2.
2. To access a level all the below levels of it should be completed. Like to play lvl3, explorer need to complete lvl1 and lvl2.
3. Each lvl is started when you click on the lvl-image. when image is clicked A form appears at the bottom of image.
4. When a level is complete press on the rightside slider on image and go to next image. Start next level by pressing on next image
5. Each form contains a story, question, clue, answer. Use clues to answer correctly. when explorer answers correctly next level unlocks and next level is played by clicking next image.
